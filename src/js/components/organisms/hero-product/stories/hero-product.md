# Hero Product

Hero product is an extention on Hero basic. The hero includes product placement and a link list.

```html
<hero-product :title="title" 
              :subtitle="subtitle"
              :description="description"
              :image="image"
              :products="products"
              :linkList="linkList"
              :ctas="ctas">
</hero-product>
```
