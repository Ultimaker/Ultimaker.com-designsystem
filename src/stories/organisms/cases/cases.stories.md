# Case

Used to highlight a single case with media like image(s) or video(s). It consists of a brief headline, text and a background image or videoloop.

It should lead to a dedicated case page or an overlay displaying the media.


## Photography guidelines

Photography of the cases image should be in the highest dimensions per breakpoint.

### Sizes full width image:
- 800 px × 450 px *(small screens)*
- 1200 px × 675 px *(medium screens)*
- 2400 px × 1350 px *(medium screens + retina)*
- 1200 px × 675 px *(large screens)*
- 2400 px × 1350 px *(large screens + retina)*
